System Backup & Restore

Note: Please close the main application before performing backup operations to prevent data errors.

Back-Up Data

Securely export your latest application data. It is recommended to perform this daily and store the backup in a safe location like an external drive. Remember, your data is irreplaceable if lost without a backup.

Restore Back-Up

Restore your application to a previous state using a backup file. Warning: This will overwrite all current data. Ensure your backup file is valid and up-to-date before proceeding.
